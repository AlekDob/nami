- [afternoon] (Alek) **fix: AI Commands text capture + replace broken on Discord/Electron** — Root cause: System Events `keystroke` doesn't reach Chromium's input handler at all — it goes through macOS accessibility layer which Electron ignores. Solution: replaced keystroke simulation with **menu-based Copy/Paste** via `click menu item "Copy" of menu "Edit"` in AppleScript. Electron apps expose standard Edit menu items that work reliably. Full strategy now: (1) Skip AX deep traversal for Electron apps. (2) Try menu Edit > Copy first (handles EN "Copy" and IT "Copia"). (3) CGEvent ⌘C fallback. (4) AppleScript keystroke last resort. Same for paste: menu Edit > Paste first, CGEvent fallback.
