- [11:50] (Alek) fix: sessions not loading in app — `String.slice(0, 100)` in `toMeta()` was cutting emoji surrogate pairs in half. Swift's JSONDecoder rejects lone surrogates, causing all 336 sessions to fail decoding silently. Added `safeSlice()` to `SessionStore`, cleaned corrupted index entry (session `de0246b0`). Also added error banner to `SessionListView` so decode errors are visible.
- [18:30] (Alek) fix: AI Command panels invisible in light mode — `NSVisualEffectView.hudWindow` follows system appearance, so in light mode the blur is light but text was forced white via SwiftUI `.environment(\.colorScheme, .dark)`. Fix: set `appearance = NSAppearance(named: .darkAqua)` on the NSVisualEffectView in all 3 panels (ResultPanel, ServicePickerPanel, QuickInputPanel). SwiftUI `.preferredColorScheme(.dark)` alone is NOT enough for AppKit layers.
